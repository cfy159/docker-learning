# docker中使用GPU硬件资源

## 安装nvidia-container-toolkit-base

**安装nvidia-container-toolkit-base**

```shell
sudo apt-get update
sudo apt-get install -y nvidia-container-toolkit-base

# 确定版本
nvidia-ctk --version
```

**生成CDI规范**

为了生成涵盖所有设备的 CDI 规范，可以使用以下命令：

```shell
sudo nvidia-ctk cdi generate --output=/etc/cdi/nvidia.yaml
grep "name:" /etc/cdi/nvidia.yaml
```

如果设备或 CUDA 驱动程序配置发生更改，则必须生成新的 CDI 规范。创建或删除 MIG 设备，或者升级驱动程序时，都可能发生配置更改。

**查看GPU列表**

```shell
nvidia-smi -L
# GPU 0: NVIDIA GeForce RTX 3060 (UUID: GPU-cf127c30-242c-f898-9f88-8fe38a225f9c)
```

## 设置Docker

```shell
curl https://get.docker.com | sh 
sudo systemctl --now enable docker
```

## 设置NVIDIA容器工具包

设置软件包仓库和GPG秘钥

```shell
distribution=$(. /etc/os-release;echo $ID$VERSION_ID) \
      && curl -fsSL https://nvidia.github.io/libnvidia-container/gpgkey | sudo gpg --dearmor -o /usr/share/keyrings/nvidia-container-toolkit-keyring.gpg \
      && curl -s -L https://nvidia.github.io/libnvidia-container/$distribution/libnvidia-container.list | \
            sed 's#deb https://#deb [signed-by=/usr/share/keyrings/nvidia-container-toolkit-keyring.gpg] https://#g' | \
            sudo tee /etc/apt/sources.list.d/nvidia-container-toolkit.list
```

```shell
sudo apt-get update
sudo apt-get install -y nvidia-container-toolkit
```

配置 Docker 守护进程以识别 NVIDIA 容器运行时：

```shell
sudo nvidia-ctk runtime configure --runtime=docker
```

设置默认运行时环境后，重启 Docker 服务以完成安装：

```shell
sudo systemctl restart docker
```

此时，可以通过运行一个基础 CUDA 容器来测试工作设置是否正常：

```shell
docker run --runtime=nvidia --gpus all nvidia/cuda:11.6.2-base-ubuntu20.04 nvidia-smi
# -e NVIDIA_VISIBLE_DEVICES： 指定GPU
docker run -it --gpus all -e NVIDIA_VISIBLE_DEVICES=GPU-1234 nvidia/cuda:11.6.2-base-ubuntu20.04 bash
```

安装`opencv`的依赖·

```shell
apt update && apt install -y libgl1-mesa-glx libglib2.0-0 libxcb* libsm6 libxrender1 libxext6
```

## 共享内存

**`shm`共享内存**是 `Linux `系统里**进程之间高速交换数据用的临时内存空间**。

`Docker` 默认给每个容器的 `shm `只有：64MB

在使用`Pytorch/TensorFlow`进行深度学习训练时，一定要调整共享内存的大小

```shell
docker run -it --gpus all --shm-size 8G nvidia/cuda:11.6.2-base-ubuntu20.04 bash
```

进入容器后查看

```shell
df -h /dev/shm
```

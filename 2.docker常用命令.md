# docker常用命令

## 镜像

```shell
# 查询本地镜像
docker images
# 删除本地镜像
docker rmi <image ID> # 一个或多个 image

# 搜索镜像
docker search <image>
# 显示 offical 的是官方镜像。

# 下载镜像
docker pull <image>
```

## 容器

```shell
# 创建容器
docker run -itd [-p <服务器端口:docker端口>] --name <容器名> <image:tag>
# -i: 交互式，保持标准输入
# -t: 为容器分配一个tty终端
# -d: 容器启动后台运行

# 删除容器
docker rm <container_id>
# 删除所有的 container
docker rm `docker ps -aq` 

# 查看容器
docker ps [-a][-q]
# -a: 该选项可以查出未启动的容器
# -q: 静默，只输出id

# 容器控制
docker start/stop/restart <container>

#进入正在运行的 docker 容器
docker exec -it [container_id] /bin/bash

# 创建软链接
docker tag <image:tag> <new_image:new_tag>
# 本地和容器文件传输
docker cp <container_id>:/<file> /target_dir
```
